# RepoMind AI

AI-powered GitHub repository analyzer built with:

- Next.js
- Express
- TypeScript
- Prisma
- Supabase (PostgreSQL)
- GitHub API
- Gemini AI (coming soon)

## Current Features

- Analyze public GitHub repositories
- Fetch repository metadata
- Backend API with TypeScript

## Roadmap

- Repository tree analysis
- AI-generated summaries
- Repository chat (RAG)
- Contributor mode
- Architecture visualization
